# Longitudinal validation of metabolic syndrome risk scores

External validation of two MetS risk scoring methods (RMRS from Shin et al. 2024 PeerJ CS, decision tree from Shin et al. 2023 PLoS One) on NHANES + NHANES Linked Mortality File. Benchmarked against PCE, Framingham 2008, and FINDRISC.

**Status:** in development.

## Reproducing

```bash
make all
```

See `Makefile` for individual phase targets.
